# プライバシーポリシー

**初版施行日: 2026-05-20**

## 1. はじめに

「ヲタ名刺 (otameishi)」(以下「本アプリ」といいます) は、推し活イベント (コミックマーケット、Vtuber イベント、ライブ、コスプレイベント等) で交換される名刺・ファン名刺を、ユーザーが個人的にアーカイブすることを目的としたモバイルアプリです。

本ポリシーは、本アプリがユーザーの情報をどのように取り扱うかを定めるものです。本アプリは「ローカルファースト」の設計思想に基づき、すべてのデータをユーザーの端末内のみで管理します。サーバーへの送信、クラウド同期、利用統計の収集等は一切行いません。

本アプリをご利用いただく前に、本ポリシーの内容を必ずご確認ください。

## 2. 取り扱う情報

本アプリは、ユーザーに関する情報を**一切収集しません**。

本アプリ内で取り扱われる以下のデータは、すべてユーザーの端末内にのみ保存され、開発者を含む第三者がアクセスすることは技術的にできません。

- 名刺情報 (名前、ハンドル、SNS アカウント、メモ等のテキスト)
- タグ情報
- イベント情報 (参加イベント名、日付等)
- マイ名刺プロフィール
- 名刺画像 (表面 / 裏面、最大 1920px / 1MB に圧縮済)
- OCR (光学文字認識) によって画像から抽出されたテキスト

### 収集しない情報

本アプリは以下のいずれも収集しません。

- 氏名、メールアドレス、電話番号等のアカウント情報 (そもそもアカウント登録機能がありません)
- 端末識別子、広告 ID
- 位置情報
- 利用ログ、アクセス解析、クラッシュレポート
- その他、ユーザーを識別可能な一切の情報

## 3. アクセス権限と利用目的

本アプリは、機能の提供のために以下の OS 権限を利用します。いずれも端末内での処理にのみ使用され、取得したデータが外部に送信されることはありません。

### 3.1 カメラ

- **利用目的**: 名刺の写真を撮影し、端末内に保存するため。また、撮影画像に対して端末内で OCR 処理を行うため。
- **送信の有無**: なし。撮影された画像および OCR 結果は端末内のみで処理・保存されます。

### 3.2 写真ライブラリ

- **利用目的**: ユーザーが既に保有している名刺の写真を本アプリに取り込むため。
- **送信の有無**: なし。選択された画像は端末内で複製・処理されるのみで、外部に送信されることはありません。

### 3.3 OCR (文字認識) 処理について

本アプリは Google ML Kit のオンデバイス文字認識機能を使用します。OCR 処理はすべて端末内で完結し、画像やテキストが外部サーバーに送信されることはありません。

## 4. 第三者提供

本アプリは、ユーザーに関するいかなる情報も第三者に提供しません。

そもそも本アプリは情報を収集していないため、提供すべき情報を保持していません。広告ネットワーク、解析サービス、データブローカー等への提供も一切ありません。

## 5. 外部送信

本アプリ自体は、いかなるネットワーク通信も行いません。

唯一の例外として、ユーザーが本アプリ内に保存された SNS リンクや Web URL を**明示的にタップした場合**に限り、OS の標準ブラウザ等の外部アプリが起動し、当該 URL にアクセスします。この通信はユーザー自身の操作によって開始されるものであり、本アプリではなく OS および外部アプリによって行われます。

外部 URL へアクセスした際の通信内容については、リンク先のサービスのプライバシーポリシーが適用されます。

## 6. データの保存場所と保存期間

### 6.1 保存場所

すべてのデータは、ユーザーの端末内、本アプリのサンドボックス領域に保存されます。

- **構造化データ**: SQLite データベースとして本アプリ専用領域に保存
- **画像ファイル**: 本アプリ専用のプライベートストレージ領域に保存

これらの領域は OS によって保護されており、他のアプリからアクセスすることはできません。

### 6.2 保存期間

ユーザーが自ら削除する、または本アプリをアンインストールするまで、データは端末内に保持されます。本アプリが自動的にデータを削除することはありません。

## 7. データの削除方法

ユーザーは、いつでも以下の方法でデータを削除できます。

### 7.1 アプリ内での個別削除

本アプリの操作により、個々の名刺、タグ、イベント、画像等を削除できます。

### 7.2 アンインストールによる完全削除

本アプリを端末からアンインストールすることにより、本アプリが保存したすべてのデータ (データベース、画像ファイルを含む) が端末から完全に削除されます。本アプリはクラウドにバックアップを保持していないため、アンインストール後にデータが復元されることはありません。

データを残しておきたい場合は、アンインストール前に必要なバックアップを取得してください。

## 8. お子様のプライバシー

本アプリは、年齢を問わずすべてのユーザーから一切の個人情報を収集しません。そのため、お子様の個人情報が本アプリを通じて開発者または第三者に提供されることはありません。

お子様が本アプリを使用される場合は、保護者の方の監督の下でご利用くださいますようお願いいたします。

## 9. 本ポリシーの変更

本ポリシーは、法令の変更や本アプリの機能追加等に応じて改定されることがあります。重要な変更を行う場合は、本アプリの更新時に変更内容をユーザーが確認できるよう告知します。

本ポリシーの最新版は、常に本ドキュメントに掲載されているものとします。
