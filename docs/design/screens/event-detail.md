# イベント詳細画面

Version 1.0

---

## 1. 目的

個別イベントの情報と紐付き名刺グリッドを閲覧する。イベントの編集・削除も行う。

---

## 2. エントリポイント

- イベント一覧の行タップ
- FAB タップ (新規作成モード)
- キャプチャレビューの「新しいイベントを作成」(モーダルで表示)

---

## 4. 状態

### 4-1. ローディング

- ヘッダー部: スケルトン
- グリッド: シマーカード × 4

### 4-2. 閲覧モード (populated)

上記ワイヤーフレーム 3-1

### 4-3. 新規作成モード

- AppBar タイトル: `event-detail.title.new`
- 全フィールド空白
- 「保存する」はイベント名入力後に活性化
- 「イベントを削除」は非表示

### 4-4. 編集モード

上記ワイヤーフレーム 3-2

### 4-5. 名刺0枚

```
│ ──── 交換した名刺 (0枚) ────  │
│                              │
│  このイベントの名刺は         │  ← type.body, color.text.secondary, 中央揃え
│  まだありません               │
```

### 4-6. 削除確認ダイアログ

```
タイトル: event-detail.delete.confirm.title
本文:    event-detail.delete.confirm.body
削除:    event-detail.delete.confirm.ok  (color.semantic.error)
キャンセル: event-detail.delete.confirm.cancel
```

削除後: イベント一覧へ (`pop`)

### 4-7. エラー (保存失敗)

スナックバーにエラーメッセージ + 「再試行」アクション

---

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| 編集 (主) | AppBar メニュー → 「編集」 | 編集モード |
| 保存 | 「保存する」ボタン | 閲覧モード |
| 削除 | 「イベントを削除」テキスト | 確認ダイアログ → イベント一覧 |
| カード詳細 | グリッドカードタップ | カード詳細画面 |
| 戻る | `arrow_back` | イベント一覧 |

---

## 6. ダーク/ライトモード差分

home.md / card-detail.md の差分パターンを踏襲。
ヘッダーパネル背景: `color.surface.secondary` (ライト: #F5F5F5, ダーク: #1E1E1E)

---

## 7. エッジケース

| 状況 | 対処 |
|---|---|
| メモなし | メモ欄を折りたたんで表示しない (編集モードでのみ表示) |
| 日付未設定 | `event-detail.date.unknown` グレーテキスト |
| イベント名50文字超 | バリデーションエラー「最大50文字です」 |
| 同名イベント | 許可。作成者が意図的に命名した可能性があるため |
| 名刺が削除された後のイベント詳細 | グリッドから自動で除外 (DB クエリが更新) |

---

## 8. アクセシビリティ

| 要素 | セマンティックラベル |
|---|---|
| ヘッダー全体 | `"コミケ106、2024年12月29日"` |
| 名刺グリッド | `"交換した名刺、38枚"` (リージョン) |
| 削除ボタン | `"イベントを削除、ダブルタップで削除確認ダイアログを開く"` |
