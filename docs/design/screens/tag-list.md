# タグ管理画面

Version 1.0

---

## 1. 目的

ユーザー定義タグの追加・確認・削除を行う管理画面。

---

## 2. エントリポイント

- 設定画面 → 「タグを管理」

---

## 3. レイアウト (ASCII ワイヤーフレーム)

```
┌─────────────────────────────┐  ← StatusBar
│ ←  タグ管理                 │  ← AppBar: type.headline
├─────────────────────────────┤
│                             │
│ ┌─────────────────────────┐ │  ← 新規タグ追加フォーム (スクロールしない固定エリア)
│ │ 新しいタグ名        [追加]│ │  ← テキストフィールド + 「追加」ボタン (一体型)
│ └─────────────────────────┘ │    フィールド: color.surface.tertiary 背景, radius.sm
│ 0 / 20                      │  ← 文字カウンター, type.caption, color.text.tertiary
├─────────────────────────────┤
│                             │  ← スクロール可能タグリスト
│ ┌─────────────────────────┐ │
│ │ [label] Vtuber     [🗑] │ │  ← タグ行, 高さ 56dp
│ │          12枚に使用中    │ │  ← type.caption, color.text.tertiary
│ └─────────────────────────┘ │
│ ─────────────────────────── │  ← 区切り線
│ ┌─────────────────────────┐ │
│ │ [label] コス        [🗑] │ │
│ │          8枚に使用中     │ │
│ └─────────────────────────┘ │
│ ─────────────────────────── │
│ ┌─────────────────────────┐ │
│ │ [label] 絵描き      [🗑] │ │
│ │          3枚に使用中     │ │
│ └─────────────────────────┘ │
│                             │
│           (スクロール)       │
└─────────────────────────────┘
```

### レイアウト仕様

- 追加フォーム: 画面上部固定、水平マージン `space.4`
- 「追加」ボタン: フィールド右端内蔵、高さ 44dp、`color.brand.primary` テキスト
- 「追加」ボタンの活性条件: 入力1文字以上 + 重複なし
- タグ行高さ: 56dp (2行テキスト対応)
- タグアイコン: `label`, 24dp, `color.brand.secondary`
- 削除アイコン: `delete_outline`, 44×44dp タッチターゲット, `color.text.tertiary`
- 区切り線: 1dp, `color.surface.tertiary`, 左マージン `space.4 + 24dp (アイコン) + space.3`
- タグはABC / あいうえお順 (文字コード順)

## 4. 状態

### 4-1. 空状態 (タグ0件)

```
┌─────────────────────────────┐
│ ←  タグ管理                 │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 新しいタグ名        [追加]│ │
│ └─────────────────────────┘ │
├─────────────────────────────┤
│                             │
│      [label アイコン 64dp]   │  ← color.text.tertiary
│                             │
│    タグがありません           │  ← type.title, 中央揃え
│   名刺を保存するときに        │  ← type.body, color.text.secondary
│   タグを追加できます         │
│                             │
└─────────────────────────────┘
```

### 4-2. ローディング

- タグ行スケルトン × 3

### 4-3. 通常表示 (populated)

上記ワイヤーフレームの状態

### 4-4. 重複エラー

```
│ ┌─────────────────────────┐ │
│ │ Vtuber            [追加]│ │  ← フィールドに error 枠線 (color.semantic.error)
│ └─────────────────────────┘ │
│  同じ名前のタグがすでにあります│  ← type.caption, color.semantic.error
```

### 4-5. 削除確認ダイアログ

```
タイトル: tag-list.delete.confirm.title  (タグ名を動的に挿入)
本文:    tag-list.delete.confirm.body
削除:    tag-list.delete.confirm.ok  (color.semantic.error)
キャンセル: tag-list.delete.confirm.cancel
```

---

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| タグ追加 (主) | 「追加」ボタン | その場でリスト更新 |
| タグ削除 | `delete_outline` アイコン (44×44) | 確認ダイアログ → リスト更新 |
| 戻る | `arrow_back` | 設定画面 |

---

## 6. ダーク/ライトモード差分

| 要素 | ライト | ダーク |
|---|---|---|
| 追加フォーム背景 | `color.surface.tertiary` (#EEEEEE) | `color.surface.tertiary` (#2A2A2A) |
| 行背景 | `color.surface.primary` (#FFFFFF) | `color.surface.primary` (#121212) |

---

## 7. エッジケース

| 状況 | 対処 |
|---|---|
| タグ名が20文字超 | 即座に20文字でトランケート |
| 重複タグ名 | 「追加」ボタン非活性 + インラインエラー |
| タグ削除後のカード | 該当タグをカードから除去 (DB CASCADE) |
| 使用中タグ削除 | 許可するが「{N}枚から削除されます」を確認ダイアログ本文に含める |

---

## 8. アクセシビリティ

| 要素 | セマンティックラベル |
|---|---|
| タグ行 | `"タグ Vtuber、12枚に使用中"` |
| 削除ボタン | `"Vtuberを削除、ダブルタップで削除確認"` |
| 追加フィールド | `"新しいタグ名を入力"` |
| 追加ボタン | `"追加、ダブルタップでタグを追加"` |
