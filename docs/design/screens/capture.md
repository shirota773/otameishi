# キャプチャ画面 (カメラプレビュー)

Version 1.0

---

## 1. 目的

名刺をカメラで撮影する。コーナー検出ガイドで名刺の位置を視覚的にフィードバックし、シャッターをタップで撮影する。

---

## 2. エントリポイント

- ホーム / カード一覧の FAB タップ
- ランチャークイックアクション「名刺をスキャン」(Phase 7)

---

## 3. レイアウト (ASCII ワイヤーフレーム)

```
┌─────────────────────────────┐  ← フルスクリーン (ステータスバー非表示)
│                             │
│   [×]            [⚡ OFF]   │  ← 左上: 閉じる (48×48), 右上: フラッシュトグル (48×48)
│                             │    color.camera.shutter (白), material symbols
│                             │
│     ╔═══════════════╗       │
│     ║               ║       │  ← コーナーガイド枠
│     ║  [名刺プレビュ]║       │    未検出: color.camera.guide (白)
│     ║               ║       │    検出済: color.camera.guide-active (ピンク)
│     ╚═══════════════╝       │  ← 4コーナーのみL字マーカー (全辺線なし)
│                             │
│  名刺を枠内に合わせてください  │  ← type.body, 白テキスト, 中央揃え
│                             │    検出時: 「名刺を検出しました」に変化
│                             │
│                             │
│                             │
│          ○        ○         │
│       (ライブラリ)           │  ← 左: フォトライブラリ選択 (MVP対象外でも枠は確保)
│                             │
│              ◎              │  ← シャッターボタン 中央配置
│         (シャッター)         │    外円: 80dp, color.camera.shutter
│                             │    内円: 64dp, color.camera.shutter-inner (ピンク)
│      space.10 下余白        │
└─────────────────────────────┘
```

### レイアウト仕様

- 全体: フルスクリーン、`color.surface.primary` 黒 (カメラ映像が全面)
- コーナーガイド: 画面幅の 75% × 名刺縦横比 (91:55 ≒ 1.65:1) の矩形
- L字マーカー: 各コーナー 20dp × 20dp, 線幅 3dp
- シャッターボタン: 画面中央下から `space.16` 上、水平中央
- フラッシュトグル: `flash_on` / `flash_off` Material Symbol, 48×48dp タッチターゲット
- 閉じるボタン: `close` Material Symbol, 48×48dp タッチターゲット

## 4. 状態

### 4-1. カメラ起動中 (loading)

```
│  (黒背景)                   │
│                             │
│      [スピナー]              │  ← CircularProgressIndicator, color.camera.shutter
│   カメラを準備中…            │  ← type.body, 白
```

### 4-2. 名刺未検出 (通常プレビュー)

- L字マーカーが白 (`color.camera.guide`)
- ガイドテキスト: `capture.guide`

### 4-3. 名刺検出済み

- L字マーカーがピンク (`color.camera.guide-active`)
- ガイドテキスト: `capture.guide.detected`
- ハプティック: 軽い振動 (HapticFeedback.lightImpact)
- アニメーション: L字マーカーが `motion.standard` 250ms でスケールアップ (1.0 → 1.05 → 1.0)

### 4-4. パーミッション拒否

```
┌─────────────────────────────┐
│                             │
│   [カメラアイコン (×印)]     │  ← color.semantic.error
│                             │
│  カメラへのアクセスが         │  ← type.title
│  必要です                   │
│                             │
│  設定からカメラのアクセスを   │  ← type.body, color.text.secondary
│  許可してください             │
│                             │
│  ┌─────────────────────┐    │
│  │     設定を開く        │    │  ← プライマリボタン
│  └─────────────────────┘    │
│                             │
│  [キャンセル]                │  ← テキストボタン, color.text.secondary
└─────────────────────────────┘
```

### 4-5. 処理中 (シャッター後)

- 画面が黒フラッシュ (`motion.instant`)
- カメラプレビュー上に半透明オーバーレイ + スピナー
- 「読み取り中…」テキスト
- ユーザー操作をすべてブロック

### 4-6. エラー状態

- 処理失敗時: スナックバーにエラーメッセージ + 「再試行」アクション
- キャプチャ画面に留まる

---

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| 撮影 (主) | シャッターボタン (80dp) | → 処理中 → キャプチャレビュー画面 |
| キャンセル | `close` ボタン (48dp) | ホーム (モーダルを閉じる) |
| フラッシュ切替 | `flash_on/off` ボタン (48dp) | その場でトグル |

---

## 6. ダーク/ライトモード差分

カメラプレビューは常に暗背景のため、ライト/ダークモードは UI トークンに影響しない。
閉じるボタン・フラッシュボタンは常に `color.camera.shutter` (白) を使用。

---

## 7. エッジケース

| 状況 | 対処 |
|---|---|
| オートフォーカス失敗 | ガイドは未検出状態のまま。撮影は可能 |
| 画面が暗い | 処理後レビュー画面で警告バナー表示 |
| 名刺が画面外 | ガイドが未検出状態 (検出されるまで待つか手動シャッター) |
| 横向き端末 | ポートレート固定 (ランドスケープは非対応) |

---

## 8. アクセシビリティ

| 要素 | セマンティックラベル |
|---|---|
| シャッターボタン | `"シャッターボタン、ダブルタップで撮影"` |
| フラッシュトグル | `"フラッシュON、ダブルタップでOFFに切替"` |
| 閉じるボタン | `"閉じる、ダブルタップでキャンセル"` |
| コーナーガイド | `excludeFromSemantics: true` (装飾的) |
| 検出アナウンス | VoiceOver: `"名刺を検出しました"` |
