# タグ管理フロー

Version 1.0

---

## フロー概要

ユーザー定義タグの作成・削除・名刺への付与を管理する。

---

## 全体ステップ図

```
[設定画面]
        │
        「タグを管理」タップ
        ▼
[タグ一覧画面]
        │
        ├── 新規タグ追加フォーム (画面上部インライン)
        │        入力 → 「追加」ボタン → タグ一覧に追加
        │
        └── 既存タグ → 右スワイプ or 削除アイコンタップ
                        └── 確認ダイアログ → 削除
```

### 名刺へのタグ付与フロー

```
[キャプチャレビュー画面] / [カード詳細 (編集モード)]
        │
        「タグを追加」タップ
        ▼
[タグ選択ボトムシート]
        │
        ├── 既存タグ一覧 (複数選択可)
        │   選択中: チェックアイコン + `color.brand.secondary` 背景
        │
        └── 「新しいタグを作成」インラインフォーム
                └── 作成 → 即選択状態で一覧に追加
        │
        「完了」ボタンでボトムシートを閉じる
```

---

## タグ作成ルール

| 項目 | 仕様 |
|---|---|
| 最大文字数 | 20文字 |
| 最小文字数 | 1文字 |
| 重複チェック | 同一名は作成不可 (大文字小文字区別なし) |
| 許可文字 | 日本語・英数字・記号全般 (改行のみ不可) |
| 最大タグ数 | 制限なし |

重複時: インラインエラー「同じ名前のタグがすでにあります」

---

## タグ削除の挙動

1. 削除確認ダイアログ表示 (`tag-list.delete.confirm`)
2. 確認後: タグレコードを削除
3. 紐付き名刺の該当タグを全件除去 (名刺は削除しない)
4. タグ一覧を更新

---

## エラー・エッジケース

| 状況 | 表示 |
|---|---|
| タグ0件 | `tag-list.empty` 空状態 |
| 1文字未満で追加ボタン押下 | ボタン非活性 (グレー) |
| 20文字超 | 入力を即座にトランケート + カウンター表示 |
| 重複タグ | インラインエラー表示 |
| タグ削除後カードのタグ欄 | 削除タグが除去された状態で表示 |
