## 3. 詳細説明 (4000 字以内)

```
推しのイベントで交換した名刺、引き出しの奥でバラバラになっていませんか？

「ヲタ名刺」は、コミケ・Vtuber イベント・ライブ・コスプレ撮影会・同人即売会などの推し活で交換した名刺を、スマホの中だけで整理できる名刺アルバムアプリです。サーバ送信もアカウント登録も一切なし。あなたの推し活の記録は、あなたの端末の中だけにとどまります。

■ こんなお悩みを解決します
・もらった名刺がカードケースや机の上でバラバラ
・どのイベントで交換したか思い出せない
・「この X のアイコンの人、誰だっけ？」が増えてきた
・写真フォルダに撮りためた名刺画像が探せない
・名刺管理アプリはビジネス向けばかりで、推し活には合わない

■ 主な機能

【表・裏 両面撮影】
名刺の表と裏を 1 枚のカードとしてまとめて記録できます。アルバムに保存済みの画像からの取り込みにも対応しています。

【OCR で文字を自動抽出】
撮影した名刺を端末内の OCR (ML Kit) が解析し、検出した文字列を「名前候補」「X (Twitter) ハンドル」「URL」に自動で振り分けます。手入力の手間を大きく減らします。

【読み取り候補パネル】
自動振り分けで余った OCR 行は「読み取り候補」として一覧表示。タップひとつで名前 / X / タグ / メモに追加できます。レイアウトが特殊な同人名刺やイラスト入り名刺でも、必要な情報だけを拾えます。

【イベントとタグで分類】
「2026 冬コミ」「◯◯生誕祭」のようなイベント単位、「絵描きさん」「コスプレイヤー」「歌い手」のようなタグ単位で、ひとつの名刺を複数の切り口に紐づけられます (多対多)。

【高速検索】
名前・X ハンドル・メモなどのキーワードと、タグ・イベントの組み合わせで横断検索。SQLite の全文検索 (FTS5) を使っているため、件数が増えても一覧表示は軽快です。

【リスト / ギャラリー切替】
情報をざっと見たいときはリスト、ビジュアルで思い出したいときはギャラリー。場面に合わせて表示を切り替えられます。

【マイカード (自分のプロフィール)】
自分の名刺情報を 1 枚登録しておけます。次のイベント前に内容を確認したいときに便利です。

【デザインを自分好みに】
6 色のアクセントカラーと、ダーク / ライト / システム連動のテーマから選べます。推しのイメージカラーに合わせるのもおすすめです。

【画像は自動で軽量化】
取り込んだ名刺画像は長辺 1920px・1MB を上限に自動圧縮。端末ストレージを圧迫しません。

■ こんな方におすすめ
・コミケ / 同人即売会で名刺を交換することが多い同人作家・参加者
・Vtuber イベントやオフ会で名刺を受け取る機会が多いリスナー
・ライブやフェスで「お見送り会」「特典会」の名刺を集めている方
・コスプレ撮影会で名刺を交換するレイヤー・カメラマン
・ビジネス向け名刺アプリでは粒度が合わず困っていた方
・クラウドに名刺データを上げたくない方

■ プライバシーについて (重要)
本アプリは「ローカルファースト」を設計の中心に据えています。

・名刺画像、OCR 結果、メモ、タグ、イベント情報など、すべてのデータは端末内の SQLite データベースに保存されます
・サーバ送信、クラウド同期、解析・テレメトリ送信は一切ありません
・アカウント登録もログインも不要です
・オフライン (機内モード) でも、撮影・OCR・検索・閲覧のすべてが動作します
・X / SNS / Web ページへの遷移は、ユーザーがリンクをタップしたときに限り、OS の標準ブラウザ等で開かれます

利用する権限は以下のとおりです。
・カメラ: 名刺を撮影するため
・写真 / メディアへのアクセス (Android 13+ は READ_MEDIA_IMAGES): 既存の画像から名刺を取り込むため

■ 想定していない用途 (今後対応予定なしの項目を含む)
・SNS 機能 / 他ユーザーとのデータ共有
・クラウド同期 / バックアップ送信
・名刺の新規作成・デザイン
・AI による画像生成
・QR コードによる名刺共有

これらは「ローカル完結・推し活アーカイブ」というコンセプトを守るため、意図的に非対応としています。

■ 最後に
推しに会えた日、誰かと名刺を交換できた日の記憶は、あとから何度でも見返したいもの。「ヲタ名刺」は、その瞬間を整理して残すための、静かで小さな相棒です。次のイベントの前に、ぜひお試しください。
```

## 4. アプリのカテゴリ提案

**第一候補: ライフスタイル (Lifestyle)**

根拠:
- 本アプリは業務上の名刺管理ではなく、推し活という「個人の趣味・生活」の記録を支援する用途である
- Google Play の「ライフスタイル」カテゴリには、コレクション管理 / 趣味記録系アプリが多数所属しており、ユーザー層 (推し活実践者) との親和性が高い
- 「仕事効率化」は業務文脈を強く想起させ、誤ったユーザー層を集めやすい
- 「ソーシャル」は他ユーザーとの交流機能が前提となるが、本アプリはローカル完結で SNS 機能を持たないため不適合

**第二候補: 仕事効率化 (Productivity)**

「整理 / アーカイブ」という側面を強調したい場合の予備候補。ただし上記理由により、初期申請ではライフスタイルを推奨。

## 6. ターゲット属性

- **年齢層**: 18 歳以上を中心に設定 (Play Console 上のターゲット年齢: 18+ を想定)。理由: 推し活イベント (コミケ、同人即売会、ライブ等) の主要参加者層に合わせるため。アプリ自体は全年齢で安全に利用可能だが、ユーザーが取り込む名刺画像の内容 (同人作品の宣伝等) には成人向け表現が混じる可能性があるため、安全側に振る。
- **興味カテゴリ**:
  - 推し活 / オタク文化 / 同人 / コミケ
  - Vtuber / 配信者 / ライブ・音楽イベント
  - コスプレ / アニメ / マンガ / ゲーム
  - コレクション管理 / 記録ツール / ライフログ
- **想定主要シーン**: イベント会場での名刺受け取り直後の整理、帰宅後の振り返り、次回イベント前の見返し
