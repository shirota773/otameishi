# イベント一覧画面

Version 1.0

---

## 1. 目的

参加イベントの一覧を表示し、イベント単位で交換済み名刺を確認・管理するエントリポイントを提供する。

---

## 2. エントリポイント

- ボトムナビ「イベント」タブ (アイコン: `event`)

---

## 3. レイアウト (ASCII ワイヤーフレーム)

```
┌─────────────────────────────┐  ← StatusBar
│  イベント                    │  ← AppBar: type.headline
├─────────────────────────────┤
│                             │  ← スクロール可能リスト
│ ┌─────────────────────────┐ │
│ │ コミケ106               │ │  ← type.title, color.text.primary
│ │ 2024年12月29日 〜 31日  │ │  ← type.caption, color.text.secondary
│ │ 38枚                    │ │  ← type.caption, color.brand.primary
│ └─────────────────────────┘ │
│                             │  ← 区切り線 1dp color.surface.tertiary
│ ┌─────────────────────────┐ │
│ │ にじフェス2026           │ │
│ │ 2026年3月15日            │ │
│ │ 12枚                    │ │
│ └─────────────────────────┘ │
│                             │
│ ┌─────────────────────────┐ │
│ │ オフ会 (推し活)          │ │
│ │ 日付未設定               │ │  ← event-detail.date.unknown
│ │ 5枚                     │ │
│ └─────────────────────────┘ │
│                             │
│          (スクロール)         │
│                             │
│                      [FAB]  │  ← `add` アイコン FAB, 右下 space.6
└─────────────────────────────┘
│ [card] [search] [event][set]│
└─────────────────────────────┘
```

### レイアウト仕様

- リスト行: 高さ 72dp 以上 (テキスト量に応じて伸長)
- 水平マージン: `space.4`
- 行内パディング: `space.4` 縦 (上下)
- 行右端: シェブロン `chevron_right` アイコン (`color.text.tertiary`, 20dp)
- イベント名: `type.title`
- 日付: `type.caption`, `color.text.secondary`
- 枚数: `type.caption`, `color.brand.primary` (ブランドカラーで視認性強調)
- ソート: 開催日降順 (最新イベントが上)、日付未設定は末尾
- FAB: `add` アイコン (イベント追加), 右下 `space.6` マージン

## 4. 状態

### 4-1. 空状態

```
┌─────────────────────────────┐
│  イベント                    │
├─────────────────────────────┤
│                             │
│       (イベントアイコン)      │  ← `event` Material Symbol, 64dp, color.text.tertiary
│                             │
│   イベントがありません        │  ← type.title, color.text.primary, 中央揃え
│  名刺を保存するときに         │  ← type.body, color.text.secondary
│  イベントを追加できます       │
│                             │
│  ┌─────────────────────┐    │
│  │   イベントを追加      │    │  ← プライマリボタン
│  └─────────────────────┘    │
└─────────────────────────────┘
```

### 4-2. ローディング

- スケルトン行 × 3 (シマーアニメーション)

### 4-3. 通常表示

上記ワイヤーフレームの状態

### 4-4. エラー

```
│   読み込めませんでした       │
│   ┌────────────────────┐   │
│   │      再試行         │   │
│   └────────────────────┘   │
```

---

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| イベント追加 (主) | FAB (`add`) | イベント詳細画面 (新規作成モード) |
| イベント詳細 | 行タップ | イベント詳細画面 |

---

## 6. ダーク/ライトモード差分

| 要素 | ライト | ダーク |
|---|---|---|
| 行背景 | `color.surface.primary` (#FFFFFF) | `color.surface.primary` (#121212) |
| 区切り線 | `color.surface.tertiary` (#EEEEEE) | `color.surface.tertiary` (#2A2A2A) |

---

## 7. エッジケース

| 状況 | 対処 |
|---|---|
| イベント名が長い | 最大2行表示、`...` truncate |
| 日付未設定 | `event-detail.date.unknown` (グレーテキスト) |
| 名刺0枚のイベント | 「0枚」と表示 (削除を促さない) |
| 同名イベントが複数 | 両方を表示 (日付で区別) |

---

## 8. アクセシビリティ

| 要素 | セマンティックラベル |
|---|---|
| イベント行 | `"コミケ106、2024年12月、38枚、ダブルタップで詳細を開く"` |
| FAB | `"イベントを追加、ダブルタップで新しいイベントを作成"` |
