# 設定画面

Version 1.0

---

## 1. 目的

タグ管理・データエクスポート・プライバシー情報・アプリ情報へのアクセスポイントを提供する。

---

## 2. エントリポイント

- ボトムナビ「設定」タブ (アイコン: `settings`)

---

## 3. レイアウト (ASCII ワイヤーフレーム)

```
┌─────────────────────────────┐  ← StatusBar
│  設定                        │  ← AppBar: type.headline (ナビゲーションなし)
├─────────────────────────────┤
│                             │  ← スクロール可能
│ ─── データ管理 ─────────────  │  ← セクションヘッダー, type.overline, color.text.tertiary
│                             │
│ ┌─────────────────────────┐ │
│ │ [label]  タグを管理  [>] │ │  ← 行高 56dp, type.body
│ └─────────────────────────┘ │
│ ─────────────────────────── │  ← 区切り線
│ ┌─────────────────────────┐ │
│ │[upload] データをエクスポ [>]│ │
│ │         JSONで出力       │ │  ← type.caption, color.text.secondary (サブテキスト)
│ └─────────────────────────┘ │
│                             │
│ ─── プライバシー ───────────  │
│                             │
│ ┌─────────────────────────┐ │
│ │[lock] プライバシー       │ │  ← 行高 auto (テキスト量による)
│ │ このアプリはデータを外部 │ │  ← type.caption, color.text.secondary
│ │ に送信しません           │ │
│ └─────────────────────────┘ │
│                             │
│ ─── アプリ ─────────────────  │
│                             │
│ ┌─────────────────────────┐ │
│ │[info] バージョン        │ │
│ │        1.0.0            │ │  ← type.caption, color.text.tertiary
│ └─────────────────────────┘ │
│ ─────────────────────────── │
│ ┌─────────────────────────┐ │
│ │[article] オープンソース [>]│ │
│ │          ライセンス      │ │
│ └─────────────────────────┘ │
│                             │
└─────────────────────────────┘
│ [card] [search] [event][set]│
└─────────────────────────────┘
```

### レイアウト仕様

- 全体: シンプルなリスト型設定 (iOS 標準設定アプリに近い構成)
- セクションヘッダー: `type.overline`、`color.text.tertiary`、水平マージン `space.4`、上マージン `space.6`
- 行高: 56dp (サブテキストなし) / auto (サブテキストあり)
- 行左アイコン: 24dp、`color.brand.secondary`
- 行右シェブロン: `chevron_right` 20dp、`color.text.tertiary` (タップ可能な行のみ)
- タップ不可行 (バージョン・プライバシー文言) にはシェブロンなし
- 区切り線: 1dp、`color.surface.tertiary`、左マージン `space.4 + 24dp + space.3`

---

## 4. 状態

### 4-1. 通常表示

上記ワイヤーフレームの状態。ローディング・エラー状態なし (ローカルのみのため即時表示)。

### 4-2. エクスポート中

「データをエクスポート」行をタップ後:

```
│ ┌─────────────────────────┐ │
│ │[upload] エクスポート中…  │ │  ← インラインスピナー (行内右端)
│ │         少々お待ちください│ │
│ └─────────────────────────┘ │
```

エクスポート完了: iOS Share Sheet / Android Share Intent を表示

### 4-3. エクスポートエラー

スナックバー: `error.generic.title` + 「再試行」アクション

---

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| タグ管理 | 行タップ | タグ管理画面 |
| データエクスポート | 行タップ | OS共有シート |
| OSライセンス | 行タップ | ライセンス一覧画面 (フルスクリーン) |

---

## 6. ダーク/ライトモード差分

| 要素 | ライト | ダーク |
|---|---|---|
| 画面背景 | `color.surface.secondary` (#F5F5F5) | `color.surface.secondary` (#1E1E1E) |
| 行背景 | `color.surface.primary` (#FFFFFF) | `color.surface.primary` (#121212) |
| セクションヘッダー | `color.text.tertiary` (#888888) | `color.text.tertiary` (#777777) |

---

## 7. エッジケース

| 状況 | 対処 |
|---|---|
| エクスポートするデータが0件 | シェアシートを表示するが空のJSONファイルを渡す |
| エクスポート対応シェアターゲットなし | OS標準の「ファイルに保存」を利用 |

---

## 8. アクセシビリティ

| 要素 | セマンティックラベル |
|---|---|
| タグを管理 行 | `"タグを管理、ダブルタップでタグ管理画面を開く"` |
| エクスポート行 | `"データをエクスポート、JSONで出力、ダブルタップで実行"` |
| プライバシー行 | `"プライバシー: このアプリはデータを外部に送信しません"` (タップ不可、role: text) |
| バージョン行 | `"バージョン 1.0.0"` (タップ不可) |
