# カード詳細画面

Version 1.0

---

## 1. 目的

1枚の名刺の全情報を閲覧する。メモを追記・編集し、SNS リンクを外部アプリで開く。

---

## 2. エントリポイント

- カード一覧のカードタップ
- イベント詳細のカードタップ
- 検索結果のカードタップ
- キャプチャレビュー保存後 (replace)

---

### レイアウト仕様

- AppBar: `color.surface.primary` 背景
- 名刺画像: 水平マージン `space.4`、縦タップで全画面ビューワー (InteractiveViewer)
- 表示名: 水平マージン `space.4`、`type.display`
- セクション区切り: `type.overline`、`color.text.tertiary`、`space.6` 上マージン
- SNSリンク行: 高さ 56dp、左端アイコン 24dp、右端 `open_in_new` アイコン、1dp ボーダー `color.surface.tertiary`
- タグチップ: タップでカード一覧に戻り該当タグで絞り込み
- メモ欄: `color.surface.secondary` 背景パネル、`radius.md`、左に 3dp `color.brand.primary` バー、プライバシーノート (`type.caption`, `color.text.tertiary`) を末尾に表示
- メモ未記入時: 「メモを追加…」プレースホルダー (タップで編集モード)

## 4. 状態

### 4-1. ローディング

- 名刺画像: シマースケルトン (同比率)
- 各フィールド: スケルトンバー

### 4-2. 通常表示 (populated)

上記ワイヤーフレームの状態

### 4-3. 編集モード

右上メニュー → 「編集」をタップ:

```
┌─────────────────────────────┐
│ ←  編集              [完了] │  ← AppBar に「完了」ボタン追加
├─────────────────────────────┤

│                             │
│ 名前                        │
│ ┌─────────────────────────┐ │
│ │ 推しさん           [✕]  │ │  ← テキストフィールド (編集可)
│ └─────────────────────────┘ │
│                             │
│ X ID                        │
│ ┌─────────────────────────┐ │
│ │ @oshi_san          [✕]  │ │
│ └─────────────────────────┘ │
│                             │
│ URL                         │
│ ┌─────────────────────────┐ │
│ │ https://example.com     │ │
│ └─────────────────────────┘ │
│                             │

│ [Vtuber ×][コス ×][+ 追加]  │
│                             │

│                             │

└─────────────────────────────┘
```

「完了」タップで保存 → 閲覧モードに戻る。

### 4-4. 削除確認ダイアログ

```
タイトル: detail.delete.confirm.title
本文:    detail.delete.confirm.body
削除:    detail.delete.confirm.ok  (color.semantic.error のボタン)
キャンセル: detail.delete.confirm.cancel
```

削除後: カード一覧へ (`pop`)

### 4-5. 画像エラー

- プレースホルダー: `color.surface.secondary` 背景 + `image_not_supported` アイコン (グレー) + `error.image.load` テキスト

---

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| 編集 (主) | AppBar メニュー → 「編集」 | 編集モード |
| 削除 | AppBar メニュー → 「削除」 | 確認ダイアログ → カード一覧 |
| SNS リンク | リンク行タップ | OS 外部ブラウザ / アプリ |
| タグタップ | タグチップ | カード一覧 (タグ絞り込み) |
| イベントタップ | イベント行 | イベント詳細画面 |
| 画像タップ | 名刺画像 | 全画面ビューワー (ピンチズーム) |
| 戻る | `arrow_back` | 直前の画面 |

---

## 6. ダーク/ライトモード差分

| 要素 | ライト | ダーク |
|---|---|---|
| メモ背景 | `color.surface.secondary` (#F5F5F5) | `color.surface.secondary` (#1E1E1E) |
| SNSリンク行ボーダー | `color.surface.tertiary` (#EEEEEE) | `color.surface.tertiary` (#2A2A2A) |
| アクセントバー | `color.brand.primary` (#E91E8C) | `color.brand.primary` (#F06292) |

---

## 7. エッジケース

| 状況 | 対処 |
|---|---|
| 表示名なし | `detail.no-name` をグレーで表示 |
| SNSリンクなし | SNSリンクセクションを非表示 |
| タグなし | `detail.no-tags` (グレーテキスト) |
| イベントなし | `detail.no-event` (グレーテキスト) + 編集モードで選択可 |
| メモなし | プレースホルダー「メモを追加…」(色: `color.text.tertiary`) |
| SNSリンクが無効 URL | リンク行を表示するが `open_in_new` タップで何もしない + トースト「リンクを開けませんでした」 |

## 8. アクセシビリティ

| 要素 | セマンティックラベル |
|---|---|
| 名刺画像 | `"推しさんの名刺画像、ダブルタップで拡大"` |
| SNS リンク行 | `"X @oshi_san、ダブルタップで外部アプリを開く"` |
| タグチップ | `"タグ Vtuber、ダブルタップでこのタグで絞り込む"` |
| メモ欄 | `"メモ: {内容}"` + `"メモは端末のみに保存されます"` |
| 削除ボタン | `"名刺を削除、ダブルタップで削除確認ダイアログを開く"` |
