# ブラウズフロー — 一覧 → フィルタ → 詳細

Version 1.0

---

## フロー概要

保存済み名刺の閲覧・絞り込み・詳細確認のフロー。完全オフライン動作。

---

## 全体ステップ図

```
[ホーム / カード一覧画面]
        │
        ├── フィルタ操作 ──────────────────────────────┐
        │   [タグチップ] タップ → タグ絞り込み          │
        │   [ソートアイコン] タップ → ソートボトムシート │
        │   [イベントバナー] (イベント詳細から流入時)    │
        │                                              │
        │                           [絞り込み済み一覧] ◄┘
        │
        ├── カードタップ ──────────────────────────────►[カード詳細画面]
        │                                                      │
        │                                              [編集・削除]
        │
        └── イベントタブ → [イベント一覧] → [イベント詳細]
                                                       │
                                               [該当カード一覧]
                                                       │
                                               カードタップ → [カード詳細]
```

---

## 各ステップ詳細

### ステップ 1: カード一覧の表示

- デフォルト表示: 保存日時の新しい順
- レイアウト: 2カラムグリッド (カードサムネイル + 表示名 + タグ)
- スクロール: 縦スクロール、無限スクロール (SQLiteから 20件ずつ遅延ロード)
- FABはスクロールダウンで縮小 (アイコンのみ)、スクロールアップで拡大

### ステップ 2: タグフィルタ

- 一覧画面上部に横スクロールのタグチップバー
- 「すべて」チップがデフォルト選択
- タグ選択: `color.brand.secondary` 背景 + チェックアイコン (複数選択可)
- 複数タグ選択時: AND条件で絞り込み
- チップバーは一覧の上に固定表示 (スクロール連動しない)

### ステップ 3: ソート

- ソートボタン (ツールバー右) タップ → ボトムシートでソート選択
- 選択肢: 新しい順 / 古い順 / 名前順 / イベント順
- 現在のソート方法は選択チェックで示す
- イベント順: イベント日付降順、イベント未設定は末尾

### ステップ 4: カード詳細への遷移

- カードタップ → カード詳細画面 (右からスライドインナビゲーション)
- 遷移アニメーション: サムネイル共有要素トランジション (`motion.enter` 300ms)

### ステップ 5: カード詳細

- 補正済み名刺画像 (フル幅) + フィールド情報
- SNSリンク: 「プロフィールを開く」ボタン → OS外部ブラウザ / 各アプリ起動
- 編集: 各フィールドインライン編集可 (フォームモードへ切替)
- 削除: 右上メニュー → 確認ダイアログ → 一覧に戻る

## イベント軸ブラウズ

```
[イベントタブ (ボトムナビ「イベント」)]
        │
        ▼
[イベント一覧] (日付降順)
        │
        タップ
        ▼
[イベント詳細] (イベント名 + 日付 + メモ + 該当名刺グリッド)
        │
        カードタップ
        ▼
[カード詳細] (ここからの遷移はバックスタック: 詳細 → イベント詳細 → イベント一覧)
```

## エラー・エッジケース

| 状況 | 表示 |
|---|---|
| カードが0件 | `home.empty` 空状態 (イラスト + テキスト + CTA) |
| タグ絞り込みで0件 | インライン「該当する名刺はありません」 + フィルタをリセットリンク |
| 画像ファイル破損 | `error.image.load` プレースホルダー + アイコン |
| 表示名なし | `home.card.no-name` をグレーテキストで表示 |
| イベントなし | `home.card.no-event` をグレーテキストで表示 |
