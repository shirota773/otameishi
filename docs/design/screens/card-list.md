# カード一覧画面

Version 1.0

---

## 1. 目的

ホーム画面と同一コンポーネント。ボトムナビ「カード」タブに対応するルート画面。home.md の仕様を正式に参照する。

---

## 2. エントリポイント

- ボトムナビ「カード」タブ (アイコン: `style`)
- カード詳細 / イベント詳細からの「戻る」
- 保存完了後の「もう1枚追加」から戻った際

---

## 3. レイアウト

home.md の ASCII ワイヤーフレームおよびレイアウト仕様を全面参照。
本ドキュメントではリストビュー (代替レイアウト) の仕様を追記する。

### 3-1. グリッドビュー (デフォルト)

home.md 参照。2カラム、カードサムネイル中心。

### 3-2. リストビュー (代替)

ツールバーのレイアウト切替アイコン (`grid_view` / `view_list`) でトグル。

```
┌─────────────────────────────┐
│  おためいし     [view_list] [sort]│
├─────────────────────────────┤
│ [すべて][Vtuber]  ...       │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │[img] 推しさん            │ │  ← 高さ 72dp
│ │ 56px  #Vtuber  コミケ106 │ │    サムネイル 56×56 (radius.sm)
│ │       2025年8月17日      │ │    type.label (名前) + type.caption (タグ, 日付)
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │[img] 名前なし            │ │
│ │ 56px  #コス              │ │
│ │       2025年8月15日      │ │
│ └─────────────────────────┘ │
│           ...               │
└─────────────────────────────┘
```

リストビュー仕様:
- 行高: 72dp
- サムネイル: 56×56dp、`radius.sm`
- サムネイル左マージン: `space.4`
- テキストエリア左マージン: `space.4`
- 区切り線: 1dp、`color.surface.tertiary`

---

## 4. 状態

home.md の「4. 状態」を参照。グリッド/リスト切替状態はローカルに保持 (アプリ再起動後も維持)。

---

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| 撮影 (主) | FAB | キャプチャ画面 |
| タグ絞り込み | タグチップ | その場で更新 |
| ソート | `sort` アイコン | ソートボトムシート |
| レイアウト切替 | `grid_view` / `view_list` アイコン | その場でトグル |
| カード詳細 | カードタップ | カード詳細画面 |

---

## 6. エッジケース

home.md のエッジケース参照。追加:

| 状況 | 対処 |
|---|---|
| タグが3件以上 | チップを2件表示 + `+N` バッジ |
| イベント名が長い | 最大15文字 + `...` |
| カード100件超 | 20件ずつ遅延ロード、スクロール末尾でローディングスピナー |
