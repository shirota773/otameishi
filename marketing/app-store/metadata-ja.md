## 4. 説明文 (4000 字以内)

```
推し活イベントで交換した名刺、引き出しの中で行方不明になっていませんか？

「あのサークルの方、お名前なんだっけ…」
「またあの現場でお会いしたいけど、X のIDが思い出せない」
「もらった名刺の束から、推しジャンルの方だけ見返したい」

ヲタ名刺 (otameishi) は、コミケ・同人即売会・Vtuber イベント・ライブ・コスプレ撮影会などで交換した名刺・推しカード・サークルカード・ショップカードを、スマホひとつで整理できる「推し活専用」の名刺アルバムです。

撮るだけで、名前や X(Twitter) のIDを自動で読み取り。イベントとタグで紐付けて、後から「あの現場の、あのジャンルの方」をすぐ呼び出せます。データはすべて端末の中だけ。サーバーに送られることも、アカウントを作る必要もありません。

──────────────────
■ 主な機能
──────────────────

【両面まるごと撮影】
名刺の表と裏、それぞれを撮影して 1 件として保存できます。手元の名刺を写真アルバムから取り込むことも可能です。

【自動文字読み取り (OCR)】
端末内蔵の文字認識で、名刺から文字列を抽出。名前候補・X (Twitter) ハンドル・URL を自動で振り分けます。AI もネットワーク通信も使わず、すべて端末内で処理します。

【「読み取り候補」で補正もかんたん】
拾いきれなかった文字列は「読み取り候補」パネルに残ります。タップひとつで、名前・X・タグ・メモのどこに入れるかを後から自由に決められます。

【イベントとタグで多重分類】
「コミケ106」「にじフェス2026」「学マス現場」など、イベントを登録して名刺を紐付け。「同担」「絵描き」「サークル」「コス」など、タグも何個でも付けられます。同じ名刺を複数のタグ・複数のカテゴリに入れられるので、自分なりの整理が崩れません。

【高速検索】
名前・X ハンドル・タグ・イベント名・メモを横断して、キーワード検索ができます。日本語にも対応。電波の届かない会場でも、過去にいただいた名刺をすぐに呼び出せます。

【リスト / ギャラリー表示切替】
ざっと一覧で確認したいときはリスト表示、写真で見返したいときはギャラリー表示。気分や用途で切り替えできます。

【マイカード】
自分のプロフィール (表示名・X・URL など) を「マイカード」として登録できます。次の現場で交換するときの控えとしてどうぞ。

【6 色のアクセントカラー + ダークモード対応】
推しカラーに寄せたり、現場で見やすい配色を選んだり。ダーク / ライト / システム連動を切り替えできます。

【画像は自動圧縮】
取り込んだ名刺画像は最大 1920px / 1MB に自動圧縮。端末ストレージを圧迫しません。

──────────────────
■ こんな方におすすめ
──────────────────

・コミケや同人即売会で、毎回たくさんの名刺をいただく方
・Vtuber イベントやライブで、ファン同士の交流が多い方
・コスプレ撮影会・オフ会で、カメラマンさんやレイヤーさんの名刺をまとめたい方
・推しジャンル別・現場別に、出会った人を整理して残しておきたい方
・SNS に投稿せず、自分だけのアーカイブとして名刺を管理したい方

──────────────────
■ プライバシーについて
──────────────────

ヲタ名刺は「完全ローカル動作」を方針としています。

・名刺画像・メモ・タグなどのデータは、すべてご利用の端末内にのみ保存されます
・アカウント登録は不要です
・サーバーへのアップロード、クラウド同期、解析データの送信は一切行いません
・通信が発生するのは、名刺に書かれた X / Web リンクをご自身でタップして開いたときだけです
・バックアップは iOS 標準の iCloud バックアップに準拠します (任意)

「もらった名刺は他人の連絡先」だからこそ、外に出さない設計にしています。

──────────────────

推し活の現場でいただいた一枚一枚を、忘れずに、なくさずに。
あのとき交換した名刺を、ずっと手元に残しておくためのアプリです。

ぜひお試しください。 (1414 字 / 4000 字)
```

## 10. 年齢制限の提案

**提案: 4+**

**根拠**

- ヲタ名刺は完全ローカルのオフラインアプリで、ユーザー生成コンテンツ (UGC) の公開・他ユーザーとの通信機能を持ちません。
- 暴力・性的表現・ギャンブル・薬物などの該当要素は一切なし。
- 外部 SNS への遷移は「ユーザーが自分でタップしたリンクを OS のブラウザ / アプリで開く」だけで、アプリ内ブラウジングや無制限ウェブアクセスは提供しません。これは Apple のレーティング基準上「無制限の Web アクセス」には該当しません。
- カメラ・写真ライブラリへのアクセスはありますが、これ自体はレーティング上の上げ要因にはなりません。

以上より、最低区分の **4+ (すべての年齢)** が妥当です。
