# v1.0.0 (MVP)

初版リリース。推し活イベントで交換した名刺をローカルで保管・分類・検索するアプリ。

## 主な機能

- 撮影 / アルバムから取り込み (両方とも OCR 実行)
- OCR 結果は自動振り分け + 「読み取り候補」パネルから手動で 名前 / X / タグ / メモ に追加可能
- 名刺の表 / 裏 両面登録
- イベントとタグでカードを分類 (多対多)
- マイカード (自分のプロフィール名刺) 登録
- ホーム: リスト / ギャラリー 表示切替
- 検索: キーワード + タグ + イベントフィルタ
- 設定: ダーク / ライト / システム 切替、6 色アクセントカラー

## 非機能

- 完全ローカル動作。クラウド同期・サーバ送信なし
- 全データは端末内 SQLite (FTS5) + アプリ専用ディレクトリの画像ファイル

## 既知の制約

- iOS 配信は未着手 (コードは対応済み、App Store メタデータ・署名・スクショが未整備)
- 名刺画像の共有機能なし (検討の結果、QR では画像が送れず、現状の MVP スコープから外した)
