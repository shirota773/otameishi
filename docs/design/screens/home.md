# ホーム / カード一覧画面

Version 1.0

---

## 1. 目的

保存済み名刺の一覧をグリッドで表示し、タグフィルタ・ソートで絞り込む。新規スキャンへのエントリポイントを提供する。

---

## 2. エントリポイント

- コールドランチ後のデフォルト画面 (ボトムナビ「カード」タブ)
- カード詳細 / キャプチャレビューから戻った後

---

### レイアウト仕様

- グリッド: 2カラム、横マージン `space.4`、カード間隔 `space.3`
- AppBar 背景: `color.surface.primary`, 高さ 56dp
- タグチップバー: `color.surface.primary` 背景、下ボーダー `color.surface.tertiary` 1dp
- FAB 位置: 画面右下、ボトムナビ上端から `space.6` 上、右端から `space.6`
- スクロール時 FAB: 縮小 (ExtendedFAB → FAB、`motion.standard`)

### 4-1. 空状態

```
┌─────────────────────────────┐
│  おためいし           [sort] │
├─────────────────────────────┤
│                             │
│                             │
│       (カードイラスト)        │

│   まだ名刺がありません        │  ← type.title, color.text.primary
│  イベントで交換した名刺を     │  ← type.body, color.text.secondary, 中央揃え

│                             │
│  ┌─────────────────────┐    │
│  │   最初の名刺を追加    │    │  ← プライマリボタン (52dp高)
│  └─────────────────────┘    │
│                             │
└─────────────────────────────┘
```

- タグチップバーは非表示 (タグが0件のため)
- FABは表示 (主要エントリポイントのため)

### 4-2. ローディング状態

- タグチップバー: スケルトン (幅 60dp × 3、`color.surface.secondary`)
- グリッド: シマースケルトン 4枚 (同比率プレースホルダー)
- FAB: 表示

### 4-3. 通常表示 (populated)

- グリッドに名刺カード
- スクロールで追加ロード (20件ずつ)
- スクロール下部に「すべて表示しました」テキスト (`type.caption`, `color.text.tertiary`)

## 5. アクション

| アクション | コンポーネント | 遷移先 |
|---|---|---|
| 撮影 (主) | FAB (`photo_camera`) | キャプチャ画面 |
| タグ絞り込み | タグチップ | その場で一覧更新 |
| ソート | `sort` アイコンボタン (44×44) | ソートボトムシート |
| カード詳細 | カードタップ | カード詳細画面 |
| 検索 | ボトムナビ「検索」 | 検索画面 |

---

## 6. ダーク/ライトモード差分

| 要素 | ライト | ダーク |
|---|---|---|
| AppBar 背景 | `color.surface.primary` (#FFFFFF) | `color.surface.primary` (#121212) |
| カード背景 | `color.surface.secondary` (#F5F5F5) | `color.surface.secondary` (#1E1E1E) |
| タグチップ (未選択) | `color.surface.secondary` | `color.surface.secondary` (#1E1E1E) |

---

## 7. エッジケース

| 状況 | 対処 |
|---|---|
| 表示名なし | `home.card.no-name` をグレー (`color.text.tertiary`) で表示 |
| タグなし | タグ行を省略してカード高さを縮小 |
| 画像破損 | グレーのカードアイコンプレースホルダー表示 |
| タグ絞り込みで0件 | 「該当する名刺はありません」+ 「フィルタをリセット」リンク |
| タグ名が長い | チップ内で最大16文字 + `...` (truncate) |
